infer module

infer.infer(img, backbone='iresnet50', pretrained=False)

The inference of arcface.

Parameters

  • img (NumPy) – The input image.

  • backbone (Object) – Arcface model without loss function. Default: “iresnet50”.

  • pretrained (Bool) – Pretrain. Default: False.

Examples

>>> img = input_img
>>> out = infer(input_img, backbone="iresnet50", pretrained="/path/to/eval/ArcFace.ckpt")
